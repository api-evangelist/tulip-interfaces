# Tulip Interfaces (tulip-interfaces)

Tulip is a manufacturing operations platform that enables frontline operations teams to build apps, connect machines and devices, and analyze production data without code. The Tulip REST API provides programmatic access to apps, tables, connectors, stations, automations, and analytics, supporting integration with ERP, MES, and industrial equipment on the shop floor.

## APIs

### Tulip API

The Tulip REST API gives programmatic access to the Tulip manufacturing operations platform, organized into namespaces covering apps, tables, table records, connectors, stations, and automations. Each namespace is independently versioned. JSON is returned by all endpoints. Authentication uses API key tokens scoped to a Tulip workspace.

- **Human URL:** [https://support.tulip.co/apidocs/introduction](https://support.tulip.co/apidocs/introduction)
